Retail Sales Data Analysis & Dashboard


📌Project Overview

This project involves end-to-end analysis of 1,000 retail transactions to uncover sales trends, customer segmentation patterns, and product performance insights. An interactive dashboard was developed using Google Sheets to support data-driven decision-making.


🔍 Key Insights

Electronics category generated the highest total revenue.

Female customers contributed 51.1% of overall sales.

Mature age group showed the highest revenue contribution.

May recorded the highest monthly sales, while September was the lowest.
